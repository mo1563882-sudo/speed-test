🔗 Techovia Mini API v1.0
Access Test Results Programmatically
Retrieve the latest speed test results directly from the browser console.
window.TechoviaAPI.results
Response Structure:
{ "ping": 0, "jitter": 0, "bufferbloat": 0, "download": 0, "stability": 0, "noise": 0, "timestamp": "2026-06-21T..." }
Usage Example:
const results = window.TechoviaAPI.results; console.log(`Download: ${results.download} MB/s`); console.log(`Ping: ${results.ping} ms`); console.log(`Noise: ${results.noise} dB`);
